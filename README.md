# Android WebView

A simple android app generator to webview based apps.

## Requirements
- Android SDK installed and available on path.
- ImageMagick

## Install
```
    $ pip install android-webview
```

## Create your app template
```
    $ android-webview 
```
Follow the steps and be happy! :)
